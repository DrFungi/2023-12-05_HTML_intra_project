Oracle Database (Oracle DB) est un système de gestion de base de données relationnelle (SGBDR) d'Oracle Corporation. Développée à l'origine en 1977 par Lawrence Ellison et d'autres développeurs, Oracle DB est l'un des moteurs de base de données relationnelles les plus fiables et les plus largement utilisés pour stocker, organiser et récupérer des données par type tout en maintenant des relations entre les différents types. Le système est construit autour d'un cadre de base de données relationnelle dans lequel les objets de données peuvent être directement accédés par les utilisateurs (ou une application frontale) par le biais d'un langage de requête structuré (SQL). Oracle est une architecture de base de données relationnelle entièrement évolutive et est souvent utilisée par des entreprises internationales qui gèrent et traitent des données sur des réseaux locaux et étendus. La base de données Oracle possède son propre composant réseau pour permettre les communications entre les réseaux. La base de données Oracle est également connue sous le nom de SGBDR Oracle et, parfois, simplement sous le nom d'Oracle.
